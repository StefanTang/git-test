## git init

## git add 文件名
把 **【工作区（实际持有文件）】** 提交到 **【暂存区】**

---
## git status 查看状态
查看当前目录状态（新增、删除、修改）

---
## git commit -m '提交的注释文本'
把 **【暂存区】** 的内容提交到 **【本地仓库】**

---
